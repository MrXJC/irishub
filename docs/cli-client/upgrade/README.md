# iriscli upgrade

## Description

Be used to participate in the software upgrade process.

## Usage

```shell
iriscli upgrade [command]
```

## Available Commands

| Name          | Description                           |
| ------------- | ------------------------------------- |
| [info](info.md)                   | Query the information of upgrade module |

## Flags

| Name, shorthand | Default | Description      | Required |
| --------------- | ------- | ---------------- | -------- |
| --help, -h      |         | help for upgrade |          |
